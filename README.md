Установка и запуск:

1. Клонировать репозиторий  git clone https://github.com/gialde/FAB7-12.git

2. Перейти в папку cd FAB7-12

3. Установить зависимости бэкенда npm install

4. Запустить бэкенд  node server.js 

5. Открыть новый терминал, перейти в папку client  cd client

6. Установить зависимости фронтенда  npm install

7. Запустить фронтенд  npm start

8. Открыть браузер  http://localhost:3001
